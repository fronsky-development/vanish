# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Vanish, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please contact the maintainer directly:

- **Email:** [support@fronsky.nl](mailto:support@fronsky.nl)
- **Website:** [fronsky.nl](https://fronsky.nl)
- **GitHub:** Open a [private security advisory](https://github.com/fronsky-development/vanish/security/advisories/new)

## Response

You can expect an initial response within **72 hours**. Critical vulnerabilities will be prioritized and patched as soon as possible.

## Scope

This policy covers the Vanish plugin, its public API, and all code within this repository.
